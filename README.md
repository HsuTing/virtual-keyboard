# Virtual Keyboard

## Start

- [Node.js](https://nodejs.org/en/) must be installed.
- Install modules:
```
  npm install -i
```

## Render

- webpack:
```
  ./node_modules/.bin/webpack
```

## Referrence

- [js-aruco](https://github.com/jcmellado/js-aruco)
